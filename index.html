    설명
    • 1.영화진흥위원회(영화예매순위 정보), NAVER(이미지 호출), 유튜브(예고편 영상제공)을 활용한 현재상영작 순위사이트입니다.
    • 2.영화진흥위원회에서 예매순위정보를 Ajax를 통해 실시간으로 가져와 상영 1위 영화예고편을 유튜브 URL로 처리하였습니다.
    • 3.NAVER는 AJAX로 외부에서 직접 이미지 정보를 가져올 수 없기 때문에, 예매순위 정보를 자체 서버로 전달한 후, 서버에서 Python을 이용하여 영화 포스터 이미지 URL을 수집하고, 이를 다시 웹페이지에 전달하여 표시합니다.
